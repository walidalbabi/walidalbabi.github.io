Support & Contact

For support with Ragdoll Creator Pro (or any Phoenix Realm Publisher tool), please contact:

Email

walid.b@phoenixrealmgames.com

Before contacting support

  • Unity version (example: Unity 6.0.x)
  • Tool version (from package/version or release tag)
  • Steps to reproduce (short + numbered)
  • Console logs and screenshots/videos if relevant
  • Whether the issue happens in a clean new project

Response time

Typical response time: 1–3 business days.

If you are a Unity Asset Store customer, include your invoice/order ID if available (optional).
