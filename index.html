Ragdoll Creator Pro

Summary: Create fully functional ragdolls for any creature — spiders, dragons, or custom rigs — with built-in pooling, LOD, and hitbox tools for optimized performance.

Unity 6+ URP / HDRP / Built-in All platforms

Overview

Ragdoll Creator Pro is a complete editor tool that brings advanced ragdoll creation to any character — humanoid or not. Unlike Unity’s built-in wizard (humanoid-focused), this tool supports spiders, quadrupeds, and custom skeletal setups with an intuitive workflow and performance-minded systems.

Key Features

  • Universal Ragdoll Builder: Works with any bone structure (not limited to humanoids).
  • Visual Setup Workflow: Configure bones, collider shapes, and joint limits in an editor window.
  • Optimized Ragdoll System: Separates ragdolls from active characters for minimal overhead.
  • Pooling Integration: Reuse ragdoll instances by spawning and syncing poses on demand.
  • Ragdoll LOD: Freeze/deactivate distant ragdolls to save CPU time.
  • Hitbox Generator: Copy collider shapes to create precise hit detection zones.
